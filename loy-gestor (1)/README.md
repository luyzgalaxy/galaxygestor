# Loy Gestor - Sistema de Gestão Financeira

Loy Gestor é um sistema avançado de gestão financeira focado no controle de empréstimos, parcelas e desempenho de carteira. Desenvolvido com React, Tailwind CSS e Inteligência Artificial.

## 🚀 Funcionalidades

- **Painel Operacional:** Controle total de empréstimos ativos, novos contratos e histórico.
- **Gestão Financeira:** Gráficos detalhados de fluxo de caixa, lucro e distribuição de carteira.
- **Central de Cobranças:** Identificação automática de atrasos e integração com WhatsApp para cobranças rápidas.
- **Bot de Cobrança:** Automação de lembretes de pagamento com logs em tempo real.
- **Análise com IA:** Insights estratégicos gerados pela IA do Google Gemini baseados nos seus dados reais.
- **Customização:** Temas claro/escuro, cores de fundo personalizadas e suporte a biometria.
- **Notificações:** Alertas visuais e sonoros para eventos críticos.

## 🛠️ Tecnologias

- **Frontend:** React 19, Vite, Tailwind CSS 4.
- **Animações:** Motion (Framer Motion).
- **Gráficos:** Recharts.
- **Ícones:** Lucide React.
- **IA:** Google Gemini API (@google/genai).
- **Markdown:** React Markdown.

## 📦 Como Rodar o Projeto

1. **Clone o repositório:**
   ```bash
   git clone <url-do-repositorio>
   ```

2. **Instale as dependências:**
   ```bash
   npm install
   ```

3. **Configure as variáveis de ambiente:**
   Crie um arquivo `.env` baseado no `.env.example` e adicione sua `GEMINI_API_KEY`.

4. **Inicie o servidor de desenvolvimento:**
   ```bash
   npm run dev
   ```

5. **Build para produção:**
   ```bash
   npm run build
   ```

## 🌐 Deploy (Netlify / Vercel)

Este projeto está pronto para ser implantado em plataformas como Netlify ou Vercel. Basta conectar seu repositório GitHub e configurar o comando de build como `npm run build` e o diretório de saída como `dist`.
